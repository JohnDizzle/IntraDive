Web Dive Browser is a fluent Windows UI3 Desktop application that is a single page web browser utilizing Google Chromium Engine. Browsing history includes information about the websites you visit with WebDive Browser. Your Settings has the ability to store your favorites, and remove them. Your history is only stored in memory and after the application is closed all application data will be erased. Notice: Chromium's Engine stores all your browsing cache, cookies, and history in windows data directory; although I have functionality to delete these collections that are stored on one's data directory(disk). This funtionality doesn't include your Search Engine's Account that one may use! 2023-01-01 John M. Doyle - Creator
